Visual Studio Code(通称VSCode)はMicrosoftが開発したエディタで、多くのエンジニアが利用している定番エディタとなっています。

VSCodeはその高度な機能性と拡張性から、コードの編集やデバッグ、Gitの統合、コードの自動補完やリファクタリングなど、幅広いタスクを効率的にこなすことができます。

その便利さを最大限に活かすためには、ショートカットキーを使いこなすことがかなり重要です。
この記事では、VSCodeで覚えておくと便利なショートカット10選を紹介します。


1. **コードの検索と置換（Ctrl + F, Ctrl + H）:** 
Ctrl + Fを押すと現在のドキュメント内を検索できます。
同様にCtrl + Hを押すと、検索したフレーズを別のフレーズに置換できます。


2. **全体の検索と置換（Ctrl + Shift + F, Ctrl + Shift + H）:** 
Ctrl + Shift + Fで全てのファイルを対象に検索し、Ctrl + Shift + Hで検索結果を一括で置換することができます。

3. **エディタ間の移動（Ctrl + 1, Ctrl + 2）:** 
VSCodeでは複数のエディタを開いて作業できます。
Ctrl + 1を押すと一つ目のエディタに、Ctrl + 2を押すと二つ目のエディタにというように複数のエディタ間を簡単に移動できます。

4. **Terminalを開く（Ctrl + J）:** 
Ctrl + Jを押すと下部に組み込みのTerminalが表示されます。
これにより、エディタを離れずにコマンドを入力することができます。
また、組み込みのTerminalがアクティブな状態でCtrl + Jを押すと、直前のエディタに戻ることができます。

5. **ファイルの閉じる（Ctrl + W）:** 
Ctrl + Wを押すと現在のエディタを閉じます。
Chromeなどのブラウザでも使えるショートカットなので覚えておいて損はないです。

6. **行の移動（Alt + ↑/↓）:** 
Alt + 上下矢印キーを使うと、現在の行を上または下に移動させることができます。
行を入れ替えたいときに重宝します。

7. **行のコピー（Shift + Alt + ↑/↓）:** 
Shift + Alt + 上下矢印キーを使うと、現在の行をコピーして新しい行を作成します。
上の行の移動と組み合わせて使うことが多いです。

8. **行の削除（Ctrl + Shift + K）:** 
Ctrl + Shift + Kを押すと現在の行を削除します。
複数行選択した状態でも有効なのでShift + 上下矢印キーで不要な複数行を選択すると、一部でも選択された行を一括で削除できます。

9. **コマンドパレットの表示（Ctrl + Shift + P）:** 
Ctrl + Shift + Pを押すとコマンドパレットが表示されます。
このパレットを使用すれば、VSCodeの機能やインストールした拡張機能の様々なコマンドを簡単に実行できます。

10. **コメントの切り替え（Ctrl + /）:** 
Ctrl + /を押すと、選択中の行をコメントアウトし、もう一度押せばコメントを解除できます。
こちらも複数行に対しても有効で、特定のコードブロックを一時的に無効にしたい場合などに役立ちます。


以上が、VSCodeで覚えておくと便利なショートカットキー10選です。
これらのショートカットはすべてデフォルトの設定ですが、好みに応じて変更することも可能です。
VSCodeの強力なカスタマイズ機能を利用して、自分だけの効率的な開発環境を作り上げていきましょう。
